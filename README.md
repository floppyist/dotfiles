## Installation

* Sway
* SwayFX 
* Waybar
* Wofi
* Alacritty
* LazyVim
* Nerd Fonts
* Noto Fonts
* Pango Fonts

### Clone repository

```console
git clone https://github.com/floppyist/dotfiles
```

### Install files

```console
mv dotfiles/.bashrc ~
```

```console
mv dotfiles/* ~/.config
```

### Cleanup

```console
rm -r ~/.config/.git
```

```console
rm -r ~/.config/screenshots
```

```console
rm ~/.config/README.md
```
